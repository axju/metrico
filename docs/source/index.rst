Metrico
=======
Metrico is a python tool to get data from social platforms like Youtube, Instagram, etc.

Disclaimer
----------
Getting data from social networks is big topic and the business model of many company's. 
Like `socialblade <https://socialblade.com/>`_,
`youscan <https://youscan.io/>`_,
`nindo <https://nindo.de/>`_,
`socialchamp <https://www.socialchamp.io/>`_,
`sproutsocial <https://sproutsocial.com/>`_ and much more.
All this service get the data from several thousand or million accounts.
Metrico is a tool to create your on analytics which can got much deeper.
